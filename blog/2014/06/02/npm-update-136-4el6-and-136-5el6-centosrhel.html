npm update from 1.3.6-4.el6 and 1.3.6-5.el6 on CentOS/RHEL

Submitted by admin on Mon, 06/02/2014 - 16:47

yum update will fail on CentOS 6.5 when upgrading between 1.3.6-4.el6 and 1.3.6-5.el6. The solution is to remove npm:

yum remove npm

Running Transaction

Error unpacking rpm package npm-1.3.6-5.el6.noarch
error: unpacking of archive failed on file /usr/lib/node_modules/npm/doc: cpio: rename
Verifying : npm-1.3.6-5.el6.noarch 1/1

Failed:
npm.noarch 0:1.3.6-5.el6

Delete the node_modules directory all together

rm -rf /usr/lib/node_modules

Then reinstall

yum install npm
